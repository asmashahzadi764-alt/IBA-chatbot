# React Website Chatbot (Frontend Only)

A **floating chatbot component** built with **React**.  
Supports **formatted responses**, **quick buttons**, **typing indicator**, and a **fully responsive design**.

## Project Structure

chatbot-frontend/

├── src/

│ ├── Chatbot.jsx # Main chatbot component

│ ├── Chatbot.css # Styling for chatbot

│ └── App.jsx # Main React app file

├── public/

│ └── index.html

├── package.json

└── README.md


## Installation

Clone the repository and install dependencies:

```bash

cd react-chatbot-frontend
npm install
Run App
Start the development server:

npm start
Open in your browser
